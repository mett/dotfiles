# dotfiles
Stored snapshots of my dotfiles


# Placements

| Source           | Destination    |
|------------------|----------------|
| nvim             | ~/.config/nvim |
| vim              | ~/.vim/        |
| tmux/tmux.conf   | ~/.tmux.conf   |
| zsh/zshrc        | ~/.zshrc       |
